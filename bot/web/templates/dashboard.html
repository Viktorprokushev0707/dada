{# ── HOW IT WORKS ─────────────────────────────────────────── #}
Как привязать группу к исследованию
  1. Создайте исследование выше (запомните его ID)
  2. Добавьте бота в Telegram-группу
  3. В группе напишите: /link ID (например /link 3)
  4. Теперь в этой группе /setup будет регистрировать участников в нужную таблицу

Одна группа = одно исследование. Чтобы переключить — повторите /link с другим ID. Для отвязки: /unlink
